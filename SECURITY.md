# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Email security concerns to the maintainers via the contact information on the [GitHub organization profile](https://github.com/mantiqjs), or use GitHub's private vulnerability reporting feature on this repository.

We will acknowledge receipt within 48 hours and aim to release a fix within 7 days for critical issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
| < 0.1   | No        |
