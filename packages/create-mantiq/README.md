# create-mantiq

Scaffold a new MantiqJS application with one command.

Part of [MantiqJS](https://github.com/abdullahkhan/mantiq) — a batteries-included TypeScript web framework for Bun.

## Installation

```bash
bun create mantiq my-app
```
